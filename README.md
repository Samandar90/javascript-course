# JavaScript с нуля / noldan — интерактивный курс

Двуязычный (🇷🇺 русский / 🇺🇿 o‘zbek) интерактивный курс для изучения основ JavaScript.

**Для учеников:** открой ссылку GitHub Pages и проходи уроки по порядку.

## Возможности
- 14 уроков: от «Что такое JavaScript» до мини-проектов
- Живой редактор кода прямо в браузере (пиши и запускай)
- Тесты после каждого урока
- Рабочие демо-кнопки (события, клики)
- Переключатель языка RU / UZ
- Сохранение прогресса в браузере

## Локальный запуск
Открой `index.html` двойным кликом.
